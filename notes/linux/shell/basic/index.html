快捷键

回到命令行开头–Home

Ctrl+a

回到命令行的尾部–End

Ctrl+e

删除光标前边的所有字符

Ctrl+u

删除光标后边的所有字符

Ctrl+k

删除光标前的一个单词

Ctrl+w

输入曾经的命令下的某个单词或字母，按照单词的匹配history

Ctrl+r
cat

在cat输出时候显示行数

cat -n maim.go
wc

统计文件行、单词、字符数量 格式：

usage: wc [-clmw] [file ...]

统计main.go的行、单词、字符数量

wc main.go

选项：

-l 统计行数
-c 统计字符数
-w 统计单词数
-L 统计最长的行的字符数
nc

简单的文件传输工具

接收方

nc -l [port] > filename

发送方

nc [ip] [port] < filename
gzip

解压*.gz的压缩文件

与*.tar.gz文件不同，*.gz文件需要用gzip来解压

gzip -d filename
